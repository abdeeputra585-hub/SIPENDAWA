Sistem Manajemen Wali Siswa

ℹ️ Informasi: Ikuti langkah-langkah di bawah untuk setup aplikasi dengan sempurna.

1Aktifkan Apache & MySQL

Pastikan XAMPP atau WAMP sudah running:

  • Buka Control Panel XAMPP/WAMP
  • Klik tombol "Start" untuk Apache
  • Klik tombol "Start" untuk MySQL
  • Status harus menunjukkan "Running" (warna hijau)

2Cek Koneksi Database

Verifikasi bahwa MySQL sudah siap:

  • Buka browser → http://localhost/phpmyadmin
  • Jika berhasil, Anda akan melihat dashboard phpMyAdmin
  • Ini membuktikan MySQL berfungsi dengan baik

3Setup Database Otomatis

Jalankan script setup untuk membuat database dan tabel:

http://localhost/[path-ke-folder]/setup_db.php

Contoh:

http://localhost/UTS%20PEMOGRAMAN/setup_db.php

Sistem akan:

  • ✓ Membuat database eduguardian
  • ✓ Membuat 6 tabel (users, siswa, wali, relasi, notifikasi, kehadiran)
  • ✓ Menampilkan status setup (SUCCESS/ERROR)

4Mulai Menggunakan Aplikasi

Jika setup berhasil, buka dashboard utama:

http://localhost/[path-ke-folder]/index.php

Anda akan melihat 3 tab:

  • 📝 Tambah Siswa - Tambah data siswa baru
  • 👨‍👩‍👧 Tambah Wali - Tambah data wali siswa
  • 🔗 Hubungkan Siswa-Wali - Buat relasi siswa dengan wali

✨ Fitur Utama Aplikasi

📝 Input Data

Tambahkan siswa dan wali melalui form yang user-friendly

📋 Lihat Data

Tampilkan daftar lengkap siswa, wali, dan relasi mereka

✏️ Edit Data

Ubah informasi siswa, wali, atau relasi kapan saja

🗑️ Hapus Data

Hapus data dengan konfirmasi untuk keamanan

⚠️ Jika Ada Masalah:
  • Error Koneksi? → Pastikan MySQL sudah aktif dan file koneksi.php benar
  • Error 404? → Cek path folder, gunakan nama folder dengan spasi %20
  • Data tidak muncul? → Pastikan sudah menambah siswa dan wali sebelum membuat relasi
  • Database sudah ada error? → Buka phpMyAdmin, hapus database eduguardian, jalankan setup_db.php lagi
⚙️ Jalankan Setup Sekarang → Buka Dashboard
